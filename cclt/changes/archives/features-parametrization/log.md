## 技术决策

| 决策 | 选择 | 放弃的方案 | 原因 |
|------|------|-----------|------|
| 参数类型 | `dict` | dataclass | 简单，无需额外依赖 |
| 默认值 | 全部保留当前硬编码值 | 优化后的值 | 向后兼容 |
| 列名生成 | `f"{func}_{param}"` | 其他格式 | 与现有命名一致 |
| 配置加载 | YAML + dict + None | 仅 YAML | 多场景灵活 |
| FeatureRegistry | 类实现 | 函数式 | 支持按需计算和元数据 |

## 知识发现

- [x] **FEATURE_PARAMS 集中管理**: 所有硬编码参数抽成字典，超参优化和消融实验可自动化
- [x] **FeatureRegistry 按需计算**: 支持按特征组名计算，避免全量计算开销
- [x] **YAML 配置驱动**: `load_feature_config()` 支持 "default"/"minimal" 等预设配置，A/B 实验基础
- [x] **大时间框架特征**: 4h/1d 特征合并到 1h 时需要偏移时间戳防泄漏 → `features.py::add_higher_timeframe_features`
